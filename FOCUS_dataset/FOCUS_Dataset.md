# FOCUS Dataset



We use DINOv2 and EfficientViT-SAM for making.


## Environmental Setting - GroundingDINO

``` bash
git clone https://github.com/IDEA-Research/GroundingDINO.git
cd GroundingDINO

pip install pip==22.3.1
pip install "setuptools>=62.3.0,<75.9"
pip install --no-build-isolation -e .
```

Download pre-trained model.

``` bash
mkdir weights
cd weights
wget -q https://github.com/IDEA-Research/GroundingDINO/releases/download/v0.1.0-alpha/groundingdino_swint_ogc.pth
cd ..

```


## Environemental Setting - EfficientViT-SAM


Download pre-trained model (efficientvit_sam_xl1.pt) from https://huggingface.co/mit-han-lab/efficientvit-sam/tree/main




## Extract Mask using DINOv2 and EfficientViT-SAM


```bash
python FOCUS_dataset/extract_mask.py
```

Before running, make sure the paths and prompt are correct.
```python
# FOCUS_dataset/extract_mask.py line19-27

CONFIG_FILE = "FOCUS_dataset/GroundingDINO/groundingdino/config/GroundingDINO_SwinT_OGC.py"
CHECKPOINT_PATH = "FOCUS_dataset/GroundingDINO/weights/groundingdino_swint_ogc.pth"
SAM_CKPT = "FOCUS_dataset/weight/efficientvit_sam_xl1.pt"


dataset_name = "mipnerf"
scene_name = "bicycle"
IMAGE_DIR = f"data/{dataset_name}/{scene_name}/images"
OUTPUT_DIR = f"data/{dataset_name}/{scene_name}/mask"
```
